# 电影发现引擎 App — 设计文档

## 概述

HarmonyOS 平台上的电影发现引擎 App，以**情绪推荐**为特色，基于 TMDB 数据的沉浸式电影探索工具。

## 数据源

- **TMDB API** — 电影信息、评分、演员、类型标签、推荐数据

## 页面结构（底部 Tab 导航）

### 首页 — 推荐流

- 顶部搜索栏（可收起）
- "你现在是什么心情？" — 情绪入口卡片，点击展开情绪选择器
  - 10 种情绪：开心 / 难过 / 解压 / 燃 / 治愈 / 刺激 / 烧脑 / 浪漫 / 放松 / 随便看看
  - 选择后情绪推荐板块置顶展示
- 今日推荐 — 大尺寸卡片左右滑动，带推荐理由文案
- 情绪推荐板块（选择情绪后出现，2-3 行横向滚动列表）
- 热门榜单 — 横向滚动（近期热门 / 高分经典 / 新片速递）

### 发现页 — 标签筛选

- 多行横向标签组：类型 / 年代 / 评分 / 地区
- 已选条件展示 + 清空按钮
- 结果网格列表（2 列）

### 我的 — 个人片单

- 想看 / 看过 / 喜欢 三个分类 Tab
- 列表 / 网格模式切换展示已收藏电影

### 详情页（独立页面，从底部推入）

- 大尺寸电影海报背景（背景模糊 + 前景剧照）
- 电影基本信息：片名 | 年份 | 片长 | 评分 | 类型标签 | 地区 | 导演 | 主要演员
- "推荐理由" 区域
- 操作按钮行：想看 | 看过 | 喜欢
- 剧情简介
- 演员表（横向滑动）
- 相似推荐（横向滑动，单击跳转）

## 核心特色：情绪推荐

用户选择情绪 → 映射到 TMDB 类型标签组合 + 评分阈值 → 返回匹配电影列表。

| 情绪 | TMDB 映射 |
|------|-----------|
| 😊 开心 | Comedy |
| 😢 难过 | Animation, Family |
| 😤 解压 | Action, Thriller |
| 🔥 燃 | Action, Adventure |
| 💭 治愈 | Drama, Family |
| 😱 刺激 | Horror, Thriller |
| 🤔 烧脑 | Mystery, Sci-Fi |
| 🥰 浪漫 | Romance |
| 😴 放松 | Documentary |
| 🎬 随便看看 | 随机高分混合 |

V1 实现：纯前端规则映射，不需要服务端。

## 推荐策略发行节奏

- **V1**：冷启动热门榜 + 标签筛选 + 情绪推荐（静态规则）
- **V2**：基于内容推荐（同导演/同类型/同演员）+ 动态推荐理由
- **V3**：可选扩展功能

## 技术要点

- HarmonyOS ArkTS 开发
- 页面级导航（Tab 切换 + 详情页栈）
- 数据持久化：用户收藏/标记状态
- 情绪推荐规则：类型映射表 + 评分筛选 + 随机化
